# Class Register (iOS)

A small, pragmatic iOS sample app for registering and listing students. The project demonstrates a simple MVVM-style organization, and basic form validation. 

**What this app does**

- Allows adding student records (name, age, gender) via the registration screen.
- Displays saved students in a table view on the data list screen.
- Performs simple input validation before saving (see `StudentDataValidation.swift`).

**Main features**

- Simple registration form with validation.
- MVVM-influenced separation: view controllers, view models, and a manager layer.

**Quick start**

1. Open the project in Xcode: open `classRegister.xcodeproj` 
2. Select a simulator or a connected device and run the app.
3. Use the registration screen to add students and view them in the data display screen.
